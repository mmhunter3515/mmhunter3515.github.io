在苹果macOS电脑使用V2Ray教程

2023-05-29 分类：v2ray-ssr-教程 评论(0)

如果你是一位iphone 、 ipad 以及macOS设备的用户且你想要在手机以及电脑还有iPad上面运行V2Ray，那就只能够选择第三方兼容的APP，目前的V2Ray还不能够直接运行客户端。

目前比较热门的软件：

  1. ShadowRocket（小火箭）

  2. 狐火

  3. Pepi

但是以上这些软件已经全部从我们国内的Apple store里面下架了，有的人会直接在网上搜索相对应的软件下载，但软件的安全性就值得考究，以下将提出这些解决方案供用户参考。

直接使用这款已经运行多年的老牌台湾科学上网商家:        羊圈机场

方案一 申请苹果美区账号

这三款产品这是在我们国内的Apple store下架，而不是在所有的iPhone手机里面消失了，因此你只需要拥有一个美国的账号就可以获取这三个软件的下载途径。

如果你有美国的朋友那么获取到美国账号的方式就很简单了，直接让你的朋友将他的账号借用给你，你退出自己手机里的账号，登录到朋友账号后，就可以在store里面下载软件了。下载完成之后再重新登回自己的ID就好。

淘宝：淘宝上有很多销售美国ID的商家，你只要找一个价钱合适的且比较有保障的店铺，下单成功之后就可以问店主要账号和密码了，我之前就是在淘宝上面买的，只花了十块钱，我觉得还是很划算的。

方案二 越狱

可以自己越狱也可以花钱让别人帮你越狱，越狱之后开启的不仅仅是这个功能还有更多的新鲜功能等着用户去发掘。

当苹果手机越狱成功之后，手机就会自带一个PP助手，我们可以直接在PP助手上面搜索Shadowrocket，然后就正常下载即可。

方案三

如果你觉得越狱会侵害到自己的手机信息安全，且没有途径获取到美国的AppleID，那么你就可以考虑使用SSR了。毕竟SSR也是非常好用的。

Shadowrocket连接V2Ray教程：

1.首先我们拿出一台苹果电脑，用Safari浏览器打开V2Ray服务器端的控制面板备用，然后浏览器就会出现下图所示的界面

2.接着我们再拿出一台苹果设备，并在设备上打开Shadowrocket，我们可以看到在页面左上角的位置有一个扫描标志，点击进入，并且将镜头对准V2Ray控制面板中的二维码扫描，当你听到滴的一声的时候，V2Ray服务器就自动添加成功了。

扫描二维码是最为简单的方式了，除此之外，还可以手动操作，那么我们就要点击页面右上方的加号了，点击后页面将转变成下方的样子，找到Vmess协议的入口之后，就直接点击进入，并将服务器IP、端口号、UUID等参数如数的填好，当然这些参数与V2Ray服务端的设置一定要是百分百一样的。

3.当服务器添加成功后，小火箭的页面里面会出现选择节点的选项，并且还会显示出当前节点的延迟速度，点击进入就可以步入科学上网的环节了。

4.全局路由的功能：全局路由有四个选项，每个选项都对应着不同的功能，以下将对这些功能进行一一的列举说明。

  • 配置：小火箭内置的代理，国内网站直连，国外常用网站走代理，这样可以保证安全性。

  • 代理：上网产生的流量都走代理的路线，这样不但可以隐蔽网络，还可以保障用户的隐私安全。

  • 直连：如果点击了直连，那么手机当前就是出于脱离小火箭的状态。

  • 场景：可以进行自定场景的定位，还可以根据当前网速来决定使用何种服务街店。

当你翻墙成功之后，在显示屏的左上方位置会有VPN的标志，希望小白朋友们看到这篇文章可以更清晰的了解科学上网。
