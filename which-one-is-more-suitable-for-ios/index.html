Shadowrocket:性价比最高的代理软件！

对于Shadowrocket十分了解的客户应该知道他的中文名字叫做小火箭，被广泛应用在iOS系统之上，自诞生之日起就收取3美元的价格，由于价格十分的低廉，得到了很多消费者们的一致青睐，成为了现如今最适合新手入手的打理软件，如果大家想要深入了解小火箭的话，大家知道它的主要优势:

  1. 拥有非常低廉的价格，在价格方面可以说是绝对王者的存在，相比于其他代理软件超过几十元的价格，小火箭确实是便宜的不得了，作为现如今性价比最高的科学上网代理软件，无论他是否成为大家整个代理过程当中的主力军，都不会在金钱方面损失什么。

  2. 支持非常全面的协议，在拥有便宜价格的同时，小火箭在协议的支持方面也非常完美，除了能够支持我们生活当中比较常见的协议之外，比如SS、SSR、Vmess、HTTP(S)、Socks5、Snell 和 Trojan 协议等，同时为了能够满足更多用户们的使用需求，还支持最为全面的混淆和加密功能，就可以解析多种形式之下的SS(R)订阅了，同时还能够兼容市面上绝大多数的机场节点，为大家提供一个精致的使用体验。而且值得大家注意的是小火箭在新协议的接纳过程也比其他代理软件好很多，代理接纳的速度非常快。

  3. 即使是一无所知的小白人物也能够轻松完成整个操作过程，作为现如今市面上最适合新手进行使用的科学上网代理软件，火箭不仅拥有十分简洁明了的操作界面，只需要简单的学习，就可以成功的运用。在具体使用过程当中，只需要大家填入机场订阅→选择相应的机场节点→随便导入一个远程规则，就可以完成最基本的操作过程了。这要比其他软件略显复杂的配置好很多，这种傻瓜式的操作过程完全能够满足各大用户的实际使用需求。

  4. 然而小火箭的优势远不只如此，能够如此火热毕定拥有很多原因，如果大家对于科学上网的定制化要求不高的话。小火箭确实是很多人的不二之选，只不过在使用过程当中也会出现一定的弊端，希望大家对不足之处进行充分的了解。比如使用过程当中的耗电量比较大、稳定性的表现相对比较平庸，远不如Surge和Quantumult X、而且不支持负载均衡、功能方面相对来说比较简单(策略组和脚本)、可定制的性能相对来说比较差、而且在使用过程当中广告比较多，去广告的能力有限和MitM不稳定等。

Quantumult X:综合表现最突出的代理软件！

Quantumult X之前的版本为Quantumult，人们常常用它与小火箭进行各个方面的对比。但是自从Quantumult X诞生以后，作为Quantumult的升级版本，不但继承了原有版本的所有功能，同时还在多个方面进行了优化，比如支持策略组和JS脚本等高级功能，并且在价格方面依然不变，永远都是八美元，并且在最初上架的几天内还可以享受到更大的优惠力度，原先的老用户可以免费领取。可以说是现如今比较实惠的代理软件了，如果想要与其他几款代理软件进行比较的话，具体表现为以下几个方面的优势:

  1. Quantumult X的主界面非常好看，拥有非常时尚的外观，非常符合现如今年轻人的审美观念，真正做到了颜值报表的存在。对策组的处理界面更是如此，完全兼顾了界面的美观和实用性，成为了不可多得的使用代理软件。为了能够方便广大用户们的使用，Quantumult X所有节点列表全部支持按配置进行分组，可以通过顶部的策略组+可自定义图标的设计，让整个使用过程变得更加方便，而且在外观上也更胜一筹，给大家带来满满的设计科技感和现代感，但是在配置过程当中的操作过程也相对来说比较复杂，毕竟更多的功能还需要具体体现出来才可以。

  2. 支持比较全面的协议，目前可以支持多种不同的协议，比如SS、SSR、Vmess、HTTP(S)和 Trojan 协议，对一点跟原先介绍的小火箭一样。虽然在整体协议方面没有小火箭那么全面，但是却可以满足大多数用户的实际使用需求。如果大家的机场仅仅只支持SSR协议的话，那么Quantumult X可以说是一个最好的选择了。

  3. 如果我们从提供的功能和价格出发的话，Quantumult X无疑是性价比之王，虽然现如今支持了很多高级功能，但是当初的售价依然没有改变，而且在功能方面要比之前的版本高出很大一部分，虽然会存在一些功能上的弊端，比如没有逻辑规则和DoH，但是普通用户完全可以顺利的使用。如果大家想要顺利的使用Quantumult X，只需要花费八美元就可以了，这样的价格非常低廉，而且在接下来的使用过程当中，不需要每年在额外支付升级性功能的用，为我们完美呈现了功能与性价比之间的最佳平衡点。

  4. 任何代理软件都不是十全十美的存在，在优质的使用过程当中也会拥有一些弊端，Quantumult X也是如此。虽然在很多用户的心目当中Quantumult X已经非常完美了，但是有些问题还是要为大家简单的介绍一下:

·新版本升级之后的Quantumult X没有完全按照Surge等常规软件的规则进行顺序的匹配设定，自己设定了一套优先级系统，就像我们生活当中的苹果手机那样，采用不同的操作系统进行使用，所以在实际规则维护当中会出现一定的问题。

·在规则类型的写法上面，Quantumult X没有采用常规的DOMAIN，而是被换做成为了host，不过大家把它写成DOMAIN也可以照常使用。同时Quantumult X并没有采用最为流行的IP-CIDR6，而是被改成了ip6-cidr，所以在使用过程当中会产生一定的麻烦。不过对于那些普通用户们来说，这点改变不会对大家的实际操作产生任何的影响，但是对于那些规则维护者们来说，可能会带来很多的麻烦。如果大家想要在一些URL标准上自己造成轮子的话，那么可能会给一些新手小白人物带来不必的麻烦。

不支持一些远程的JS脚本，虽然这个问题在基本上已经得到了解决，但是在日后的运行过程当中作者可能还会出现一定的改动，这一点需要大家提前的了解。就目前而言Quantumult X仅仅只支持TF版本之下的JS脚本，商品版本在v1.0.10之后又把远程版本加了回来，如果大家在使用了一段时间之后，脚本就不能直接作为远程资源进行缓存了，这样在使用过程当中还是比较麻烦。如果那些特殊用户拥有JS脚本方面的实际需求，可能Quantumult X就没有我们想象当中的那么实用了，可能还需要借助WorkingCopy 等软件采用本地挂在脚本库的方法才能使用，也可以选择直接使用Surge或者Loon等。

·Quantumult X不太适合于新手进行使用，为了能够给用户们提供更多的功能，Quantumult X在使用过程当中隐藏了很多细节，使得上手难度相对来说比较大，特别适用于那些使用高级功能的用户们，如果仅仅是想通过最简单的方法进行使用，那么Quantumult X显然无法实现。不过Quantumult X已经成为了iOS平台上最为推荐的代理软件之一，如果大家想要使用一些高级功能，又不想支付一个比较高昂的使用费用，那么Quantumult X无疑是最好的选择了。

Surge:大佬级别的存在！

Surge可以说是最强大的代理软件了，同时在iOS平台上也拥有举足轻重的地位。主要还是因为拥有优秀的稳定性和出色的软件设计，甚至还可以为用户们提供强大的高级功能，只不过在价格方面相对来说比较高昂，第一年销售的价格超过50美元，后续升级还需要大家支付另外的费用，所以在价格方面还是比较高，这也是因为足够优秀所导致，接下来就为大家解答一下他的优秀之处。

在目前市面上所有科学上网代理客户端当中，Surge支持iOS和macOS的时间相对来说比较早，并且在两个平台上所支持的功能相差无几，也就是在macOS系统之上多了一个网关模式罢了，但是对大家的实际使用过程并不会造成很大的影响，也可以使用同一份配置软件，从而让整个使用过程变得更加简单。虽然我们仅仅只讨论iOS系统的相关内容，但是Surge对于macOS也算是一款功能比较强大的代理软件了。

可以为广大用户提供最为全面的规则支持服务，除了能够满足其他代理软件支持的规则之外，比如DOMAIN, DOMAIN-SUFFIX, DOMAIN-KEYWORD, IP-CIDR, USER-AGENT 等，还可以支持URL-REGEX, 逻辑匹配 (AND, OR, NOT 三种) 等，这是其他代理软件无法比拟的巨大优势。如果我们从详细的规则集方面出发，Surge能够拥有如此巨大的优势，主要得益于推出的时间相对来说比较早、使用的人数相对比较多、规则支持相对比较好、拥有对丰富的在线规则集等，能够满足现如今大多数用户的实际使用需求。而且对于其他热门规则仓库也拥有完整的支持力度，比如lhie1/Rules、神机规则等，高级玩家们只需要自定义规则进行编写使用就可以了。

可以支持比较完善的JavaScript 脚本，同时也是目前这个脚本做的最好的代理软件了，完全要比其他代理软件好很多。如果大家在使用过程当中想要使用远程脚本，并且能够拥有去广告和自动签到等高级功能的话，Surge绝对可以说是一个最好的选择了。

Surge除了能够支持双平台和相应的规则系统及脚本之外，还有一些值得大家关注的高级特性，这也是他如此高昂价格背后的主要支撑点，需要大家对这些内容进行充分的了解。

拥有比较先进的最近请求/抓包功能。可能Surge扬言自己是一款网络测试工具，其实这只是挂羊头卖狗肉的一个操作，主要是为了能够有效避免相应的网络审查制度，这也是为了能够给用户们提供更加完美的服务。所以他会拥有一个比较完美的抓包功能，虽然其他软件也可以提供抓取并保存的相关请求，但是Surge无论是在界面还是易用性能上面都做的十分优秀，能够让整个过程变得更加简单明了，从而方便大家的实际使用过程。

SurgeiOS在4.3.0版本以后，要完全可以支持更高级的 DNS over HTTPS (DoH) ，在这个功能方面还是比较先进的。目前我们生活当中大多的代理软件，也只有该软件提供了这样一个功能。如果大家现在正在使用Surge，那么完全不要忘记开发一下更高级的功能，比如安装 Module 来体验阿里公共 DNS 的 DoH 服务，从而享受到更加优质的服务。

支持最新的Modul功能。可能很多人对于Modul并不是十分的理解，Modul是一系列设置的集中表现，在实际使用过程当中可以覆盖当前配置的部分设定，如果大家对于实际操作过程不了解，就可以直接安装他人写好的Modul，这样就可以一键搞定了。而且为了能够给大家提供一个良好的使用体验，还可以提供去除特定广告的高级功能，能够免除大家被广告侵扰的麻烦。

如果大家是一些比较高端的玩家，Modul同样可以为大家进行微调和开关部分的设置，让整个使用过程变得更加简单。只要大家能够顺利的使用Modul，就可以轻松实现开关 WiFi Access、开关 DoH、开关对所有域名进行 MitM 等功能，大家可以根据自己的使用体验进行相应的选择。

所以在很多人的心目当中，Surge就是目前为止iOS系统平台上最强大的科学上网代理软件了，没有之一，可以为用户们提供数量庞大的创新性高级功能。而且对于现如今的一些创新性高级功能来说，其实好开始都是由Surge一手创建而来，只不过这些全新的功能会被其他代理软件模仿，这也是一件比较难以解决的事情。

Surge主要优势就在于这些先进的高级功能，如果大家对于这些高级功能没有实际性的需求，面对如此高昂的费用，Surge还是非常不实用的存在。当然，Surge并非十全十美的存在，也存在一些不足之处需要大家了解:比如不支持SSR协议、一次购买完成只能够供3台设备进行使用、如果是iOS系统和Mac系统进行使用的话，那么在使用过程当中的购买成本将超过100美元，这一点还需要大家进行充分的了解，并合理安排自己的资金。

Loon:一款潜力无限的代理软件！

市面上除了拥有我们生活当中比较常见的几款在软件之外，也拥有很多新晋代理软件，比如Loon。Loon目前还处于研发阶段，所以在实际使用过程当中还有许多不完善的地方，依旧能够满足大家平常的使用功能性，而且在一定程度上符合Surge和Quantumult X的相关优点，未来将会拥有无限的发展潜力，

现如今很多年轻人都非常关注代理软件的颜值情况，而Loon的颜值完全值得肯定，按照Surge进行设置，能够让整个操作变得更加简单，主要表现为以下几方面的优势:

  1. 支持多种不同的协议，比如SS、SSR、Vmess等，大家可以根据自身实际情况进行相应的选择。

  2. 支持一键导入Surge Ruleset的功能，方便大家对规则进行编写。

  3. 可以支持Surge的策略组写法。

  4. 可以支持多种不同的订阅节点，支持内容与Quantumult X相类似，比如 Rewrite (rewrite_remote)。

  5. 支持一定的远程脚本。

  6. 可以对本地节点进行过滤，这个过程不需要借助API就可以完成。

我能根据大量数据研究表明，如果Loon在未来还能够继续保持在功能方面的更新和维护，那么Loon将无疑成为Quantumult X最为强进的对手，甚至能够一跃成为我们生活当中最好用的代理软件，我们一起期待吧。

为了能够方便大家对以上四款代理软件进行细致的区分，我们还需要从多个方面进行对比比较，这样能够方便大家对相关内容进行充分的理解，接下来就将Shadowrocket/Quantumult X/Surge/Loon对比推荐列表送给大家:

  3. Surge：支持的平台有iOS/MacOS、支持的协议有SS/VMess/Trojan/Snell、支持的策略组规则支持，较为复杂、操作难度困难、价格为49.9美元，升级新功能19.9美元/年7.99美元。
